A photo booth implemented in NodeJS using web sockets on the communication layer. A user can connect to the photo booth on his phone, use it as a remote trigger, then get the images pushed to their phone.

Triangles in Space

Open source, multi-player space shooting game inspired by the classic "Asteroids" game. I used nodeJS for the game server and its hosted on Digital Ocean using PM2 to do node process management.
